## Folder structure

lab\_{_index_}:

- data
- task.ipynb
- main.py

utils:

- pandas operations
- plot operations
- etc

## Running and Testing

build a virtual environment, open a terminal and type:\
`python -m venv venv` \
Activate your virtual environment:\
**Linux / Mac**: `source venv/bin activate` \
**Windows**: `.\venv\bin\activate`
